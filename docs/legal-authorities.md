# Legal Authorities — TX/NM Municipal Powers

**Authority:** TX Gov't Code Chapter 552
**Applies to:** City of El Paso, EPWater (as public entity)

### Municipal Utility Regulation

### Development Agreements
**Authority:** TX Local Gov't Code §212.172 (development agreements)

## New Mexico

### Open Records (NM IPRA)
**Authority:** NM Stat. §14-2-1 et seq.
**Applies to:** Doña Ana County

### Municipal Powers
**Authority:** NM Stat. §3-18-1 (general municipal powers)

## Key Powers

**Cities CAN:**
- Require development agreements with performance terms
- Establish utility service conditions
- Request cost-of-service studies for rate structures
- Set zoning and land use conditions

**Cities CANNOT (without statutory authority):**
- Set utility rates (if utility is independent authority)
- Impose retroactive penalties
- Take property without compensation
- Regulate beyond municipal boundaries
