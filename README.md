# Civic Ledger — El Paso Proof Engine

**Production-ready CustomGPT for proof-based civic accountability**  
Production-ready CustomGPT for proof-based civic controls on data center utility impacts in El Paso, TX and Doña Ana County, NM. Generates policy documents, records requests, and accountability frameworks. Built by [Celaya Solutions](https://celayasolutions.com)

---

## Purpose

Generate **verifiable civic control frameworks** that protect El Paso residents from ratepayer cross-subsidy related to data center utility impacts.

**Key principle:** Pro-growth AND pro-fairness. Design controls so fairness is measurable, not assumed.

---

## What This Tool Generates

| Deliverable | Audience | Format |
|------------|----------|--------|
| **Staff checklists** | City/County officials | 1-page implementation guide |
| **Resident explainers** | General public | Plain language FAQ |
| **Records requests** | Advocates/journalists | TX PIA / NM IPRA templates |
| **Policy frameworks** | Council/Commission | Ordinance/resolution skeletons |
| **Technical specs** | IT/utility staff | Data schema, API endpoints |

All outputs include:
- ✅ Cause → Effect analysis
- ✅ Feasibility checks
- ✅ Source citations
- ✅ Legal review flags
